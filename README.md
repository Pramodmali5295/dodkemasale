# Dodke Masale - Authentic Indian Spices

Official website for Dodke Masale, featuring their legacy, premium spice products, and traditional manufacturing excellence.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS (Shadcn UI)
- GSAP (Animations)
- Lucide React (Icons)

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run developmental server: `npm run dev`
4. Build for production: `npm run build`
